## Purpose

Submit and/or reset a form, using links/buttons that are outside the child DOM of the form.


## [Actions](https://stimulus.hotwire.dev/reference/actions)

| Action | Purpose |
| --- | --- |
| `reset` | Reset the form, using `form.reset()` |
| `submit` | Submit the form, using `form.submit()` |

## [Targets](https://stimulus.hotwire.dev/reference/targets)

| Target | Purpose | Default |
| --- | --- | --- |
| `form` | The form to target the controller's actions on | - |

## Events

| Event | When | Dispatched on | `event.detail` |
| --- | --- | --- | --- |
| `submit` | When the form is submitted, via the `submit` action | The `form` target. | - |
| `reset` | When the form is reset, via the `reset` action | The `form` target. | - |
