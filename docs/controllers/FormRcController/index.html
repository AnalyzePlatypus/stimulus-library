FormRcController

Purpose#

Submit and/or reset a form, using links/buttons that are outside the child DOM of the form.

Actions#

ActionPurpose
resetReset the form, using form.reset()
submitSubmit the form, using form.submit()

Targets#

TargetPurposeDefault
formThe form to target the controller's actions on-

Classes#

ClassPurpose
--

Values#

ValueTypeDescriptionDefault
----

Events#

EventWhenDispatched onevent.detail
submitWhen the form is submitted, via the submit actionThe form target.-
resetWhen the form is reset, via the reset actionThe form target.-

Side Effects#

None

How to Use#

HTML#

<div data-controller="form-rc">
<form id="test-form" data-form-rc-target="form">
<div>
<label>
Name
<input type="text" name="name"/>
</label>
</div>
<div>
<label>
Age
<input type="number" name="age"/>
</label>
</div>
<div>
<label>
Quest
<textarea name="quest"/>
</label>
</div>
<div>
<label>
Do you like cake?
<input type="checkbox" name="gorilla"/>
</label>
</div>
</form>
<p>Outside the form</p>
<div>
<a href="" data-action="form-rc#reset">Reset</a>
</div>
<div>
<a href="" data-action="form-rc#submit">Submit</a>
</div>
</div>

HAML#

%div{ data: { controller: "form-rc"}}
%form#test-form{ data: {form_rc_target: "form"}}
%div
%label
Name
%input{name: "name", type: "text"}
%div
%label
Age
%input{name: "age", type: "number"}
%div
%label
Quest
%textarea{name: "quest"}
%div
%label
Do you like cake?
%input{name: "gorilla", type: "checkbox"}
%p Outside the form
%div
%a{ href: "", data: { action: "form-rc#reset"} } Reset
%div
%a{ href: "", data: { action: "form-rc#submit"} } Submit
